# SponsorConnect

A youth sports sponsor matching MVP that connects local businesses with youth sports programs.

## Features

- **Sponsor Listings** - Browse available sponsors and their offerings
- **Program Profiles** - View youth sports programs and their needs
- **Match Suggestions** - AI-powered matching between sponsors and programs

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS v4

## Getting Started

```bash
cd sponsor-connect
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Built files are in the `dist` directory.
